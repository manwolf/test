Nothing will be done if the cache already contains the key.

 Method mget (line 61)
 
 

array mget(
array
$ids)


	

Retrieves multiple values from cache with the specified keys.


Some caches (such as memcache, apc) allow retrieving multiple cached values at one time,  which may improve the performance since it reduces the communication cost.  In case a cache doesn't support this feature natively, it will be simulated by this method.

		
Parameters

	
    
			
  • array $ids: list of keys identifying the cached values
    • 
		

		
	
Info

	
    
			
  • return - list of cached values corresponding to the specified keys. The array  is returned in terms of (key,value) pairs.  If a value is not cached or expired, the corresponding array value will be false.
    • 
			
  • access - public
